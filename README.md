# 🎲 旅游地点抽签 - 广州出发5天自驾游

一个有趣的旅游目的地抽签工具，帮你决定下一次旅行去哪里！

## ✨ 功能特点

- 🎯 **三步抽签**：省份 → 城市 → 详细攻略
- 🗺️ **覆盖范围**：8个省份，50+个城市
- 📅 **详细攻略**：热门城市含5天4夜完整行程规划
- 💰 **预算明细**：交通、住宿、餐饮、门票全包含
- 📱 **移动适配**：支持手机、平板访问

## 🎮 使用方法

1. 打开网页
2. 点击"开始抽签"抽取省份
3. 再次点击抽取城市
4. 查看详细旅游攻略

## 🌟 热门目的地

已包含超详细5天4夜攻略的城市：
- 深圳 - 现代都市之旅
- 珠海 - 浪漫海滨度假
- 桂林 - 山水甲天下
- 厦门 - 文艺小清新
- 三亚 - 热带海岛天堂
